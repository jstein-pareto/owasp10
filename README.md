# OWASP Top 10 - 2017

* description: The Ten Most Critical Web Application Security Risks
* organization: [Open Web Application Security Project](https://www.owasp.org)

---

## Start

1. Install

    ```shell
    npm i
    ```

2. Run

    ```shell
    npm run start
    ```

3. Visit

    ```shell
    http://localhost:8000
    ```

4. Other Commands

    ```shell
    npm run build
    ```

    ```shell
    npm run serve
    ```

    ```shell
    npm run clean
    ```
